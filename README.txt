== Description ==

Quick Web Notes transforms how you capture and manage ideas in WordPress. Whether you're writing content, managing tasks, or brainstorming, this plugin provides a seamless note-taking experience from both the frontend and backend of your website.

= Key Features =

* **Frontend Note Creation**: Add notes directly from any page without accessing the dashboard
* **Drag-and-Drop Positioning**: Place the notes icon anywhere on your screen
* **Comprehensive Dashboard**: Manage all notes from a dedicated admin interface
* **Role-Based Access**: Control who can create, edit, and view notes
* **Real-Time Updates**: Notes appear instantly without page reload
* **Mobile-Friendly**: Fully responsive design works on all devices
* **Multiple Note Types**: Categorize notes as tasks, reminders, or ideas
* **Quick Search**: Find notes easily with instant search functionality
* **Bulk Actions**: Manage multiple notes simultaneously
* **Custom Styling**: Match the notes' appearance to your theme

== Installation ==

1. Upload the 'quick-web-notes' folder to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Settings > Quick Web Notes' to configure plugin options
4. Position the notes icon on your frontend by dragging it to your preferred location
5. Start creating notes!

= Minimum Requirements =

* WordPress 6.2 or higher
* PHP 7.4 or higher
* MySQL 5.6 or higher

== Frequently Asked Questions ==

= How do I create a note from the frontend? =

Click the floating notes icon visible on your website, enter your note content, and click save. The note will be instantly created and stored.

= Can I organize notes into categories? =

Yes! You can assign types (task, reminder, idea) to your notes and filter them accordingly in the dashboard.

= Is there a limit to how many notes I can create? =

No, you can create as many notes as you need. However, we recommend regular cleanup of completed or outdated notes for better organization.

= Can I change the appearance of the notes icon? =

Yes, you can customize the icon's size, color, and style through the plugin settings.

= How do I control who can create and view notes? =

In the plugin settings, you can assign note creation and viewing permissions based on WordPress user roles.

= Does this plugin slow down my website? =

No, Quick Web Notes is optimized for performance. It loads resources only when needed and uses WordPress best practices for efficiency.
